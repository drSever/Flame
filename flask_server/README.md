# Local Installation

### 1. Clone the repo
$ git clone https://github.com/drSever/Flame.git

$ cd Flame/flask_server

### 2. Install Python packages
$ pip install -r requirements.txt

### 3. Run
$ python app.py

### Open http://127.0.0.1:5000
